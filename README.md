# 博客來電子書下載工具

在博客來電子書閱讀器內直接下載並合併成 PDF。

---

## 安裝方式

### Tampermonkey 腳本

1. 安裝 [Tampermonkey](https://www.tampermonkey.net/)
2. 開啟 `chrome://extensions`（或 `edge://extensions`）
3. 右上角開啟「**開發人員模式**」
4. 找到 `篡改猴` 點擊 `詳細資料` 按鈕
5. 開啟 `允許使用者指令碼`
3. 前往 [Greasy Fork](https://greasyfork.org/zh-TW/scripts/582667-%E5%8D%9A%E5%AE%A2%E4%BE%86%E9%9B%BB%E5%AD%90%E6%9B%B8%E4%B8%8B%E8%BC%89%E5%B7%A5%E5%85%B7) 點「安裝此腳本」

### Chrome 擴充套件

1. 下載並解壓縮 `extension.zip`
2. 開啟 `chrome://extensions`（或 `edge://extensions`）
3. 右上角開啟「**開發人員模式**」
4. 點「**載入未封裝項目**」，選擇解壓縮後的資料夾

---

## 使用方式

1. 開啟博客來電子書閱讀器，點右下角的書本圖示開啟面板（擴充套件也可改點瀏覽器工具列圖示）
2. **整本下載**：確認書名與總頁數後，點「下載整本 PDF」
3. **分章節下載**：切換至「分章節下載」，在閱讀器點開目錄面板後目錄將自動載入，勾選要下載的章節，點「下載所選章節」

---

## 注意事項

- 僅限已購買並有閱讀權限的書籍
- 本工具僅供個人備份用途，請勿用於商業用途或違反相關法律規範
